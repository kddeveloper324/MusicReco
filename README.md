## üéß Music Recommendation System

This project is a content-based music recommendation engine designed to suggest songs similar to a user's selection. It features a modern, dark-mode user interface for an immersive experience.

### ‚ú® Key Features

* **Easy Search:** Simple two-step process to select an **Artist** and a specific **Song** from the database.
* **Visual & Detailed Results:** Displays high-quality **album artwork**, song duration, and star ratings for the selected track.
* **Smart Recommendations:** Generates a "You might also like..." list containing similar tracks (e.g., suggesting *The Doors* based on *ABBA*).
* **Spotify Integration:** Includes a **"Listen on Spotify"** button for immediate playback of recommended tracks.
* **Recursive Discovery:** Features a **"Similar"** button next to every recommended song, allowing users to dive deeper and find recommendations based on the new results.
